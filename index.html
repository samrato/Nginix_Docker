Welcome! a simple portfolio Showing Nginx

This project demonstrates how to build and deploy a secure and scalable web application using modern web technologies. A simple webpage is served by Node.js, containerized using Docker, and deployed behind an Nginx reverse proxy. The proxy not only balances load between multiple backend containers but also secures communication with HTTPS using a self-signed TLS certificate. This setup mimics real-world web infrastructure, providing a strong foundation for production-ready deployments.
